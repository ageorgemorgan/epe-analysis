# EPE Analysis

## Name
EPE (extreme precipitation event) Analysis 

## Description
Simple Python tools to help analyze extreme precipitation events simulated by Earth System Models, with a focus on CanAM. 

Also used to store code for reproducing figures in the preprint "Dominance of Resolved over Parameterized Precipitation during Extreme Precipitation Events in the Canadian Atmospheric Model" (A. G. Morgan, N. F. Tandon). For readers of this paper: the names of the .ipynb files in this repo will guide you to whatever figure you want to reproduce. Before running any of the notebooks, please make sure you have configured and activated a virtual environment using `requirements.txt`. Also, make sure you have downloaded the data from the link in the paper.
